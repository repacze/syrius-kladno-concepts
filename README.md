# Syrius Kladno — tři webové koncepty

Moderní redesign původního webu Pohřebního ústavu Syrius v Kladně.

- **Nocturne** — klidný, prémiový a důstojný
- **Lumen** — světlý, lidský a přístupný
- **Editorial** — výrazný, typografický a současný

Všechny varianty jsou responzivní, statické a připravené pro GitHub Pages.
